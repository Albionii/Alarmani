# Alarmani
Ky është projekti im i parë në gjuhen programuese Java. Është një "Alarman"(Alarm-Man) që tregon sa është koha në kompjuterin tuaj me anë të një shigjete. Këtë projekt e kam përfunduar me datën 17/2/2021.
